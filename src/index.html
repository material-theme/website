Material Theme

A free minimalist theme for Visual Studio Code & more inspired by Material Design System.

Get for VS Code Check on Github

Overview

With Material Theme your dev experience will get a boost and your eyes will be always grateful to you.

Born originally for Sublime Text it is now one of the most used dark theme for VS Code with a lot of portings to other tools.

Features

  • A lot of supported languages.
  • Five theme variants.
  • File & folders icons included.
  • Distractions free.
  • Available for other tools.
  • Constant updates.
